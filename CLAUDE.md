# Claude Code Project Guidelines & Permissions

## File Access & Tools
- You have FULL read, write, edit, and command execution access in this codebase.
- When asked to edit, optimize, or build code, DO NOT give generic advice. Use your `Edit`, `Write`, and `Bash` tools to inspect and modify the files directly.

## Stack & Architecture
- Framework: Next.js (App Router) + React 19 + TypeScript
- Styling: Tailwind CSS v4 + Vanilla CSS
- 3D & Animation: Three.js, @react-three/fiber, @react-three/drei, GSAP

## Commands
- Dev Server: `npm run dev`
- Build: `npm run build`
